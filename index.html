Learning Roadmap

Your step-by-step journey from where you are now to where you want to be.

PHASE 0 · Month 1-2

🏗️ Foundation Building

Start with the absolute basics: how computers work, how the internet works, basic Linux commands, and simple Python scripting. This is non-negotiable — you cannot secure what you don't understand. Don't rush this phase.

⏱ 8–10 weeks
PHASE 1 · Month 3-4

🔐 Core Security Concepts

Learn the CIA Triad, cryptography basics, authentication systems, types of attacks, and how defenses work. Understand OWASP Top 10. Set up a home lab with VirtualBox — this is where you start doing, not just reading.

⏱ 8 weeks
PHASE 2 · Month 5-6

🌐 Networking & Web Security

Deep dive into networking protocols (TCP/IP, DNS, HTTP/HTTPS). Learn to use Wireshark, Nmap, and Burp Suite. Practice basic web vulnerabilities in safe environments like DVWA, TryHackMe, and HackTheBox.

⏱ 8 weeks
PHASE 3 · Month 7-8

💻 Ethical Hacking Basics

Learn penetration testing methodology. Practice enumeration, exploitation, and post-exploitation in labs. Understand the legal and ethical boundaries. Start building a portfolio of write-ups (CTF solutions). Aim for CompTIA Security+ at this stage.

⏱ 8 weeks
PHASE 4 · Month 9-10

🛡️ Defensive Security (Blue Team)

Learn SIEM tools (Splunk, ELK Stack), SOC operations, incident response procedures, and digital forensics. Blue team skills are HIGHLY valued in government jobs. Learn about CERT-In guidelines and reporting.

⏱ 8 weeks
PHASE 5 · Month 11-12

🤖 AI in Cybersecurity

Learn how AI/ML is used for threat detection, anomaly detection, automated penetration testing. Understand prompt injection, LLM security risks, and how to secure AI systems. Build AI-powered security tools using Python + APIs.

⏱ 8 weeks
PHASE 6 · Month 13-18

🎓 Advanced Specialization

Choose your track: Red Team (OSCP), Blue Team (CISSP/CISM), Cloud Security (AWS/Azure security), or Malware Analysis (GREM). Build real projects. Contribute to bug bounty programs. Apply for internships and government positions. Start planning your business.

⏱ 6 months+
PHASE 7 · Ongoing

🚀 Professional & Business

Write technical blog posts. Speak at events. Network with professionals on LinkedIn. If pursuing government — prepare for UPSC/SSC CGL/NIC/DRDO exams. If business — register a company, get empaneled with government agencies, and build a team.

⏱ Continuous

AI in Cybersecurity

The future belongs to those who can combine human expertise with AI capabilities. Here's how AI is transforming this field.

⚡ The Reality Check — What AI Changes

AI will NOT replace skilled cybersecurity experts — it will replace mediocre ones. AI automates repetitive tasks (log analysis, basic vulnerability scanning, report writing). This means the bar for "good enough" rises. You need to be an expert who uses AI as a superpower, not someone who fears it. The most valuable skill in the next 10 years: AI-augmented security expertise.

🛡️
AI for Threat Detection
Machine learning models that detect anomalies and threats faster than any human can.
  • Behavioral analysis with ML (UEBA)
  • Anomaly detection with autoencoders
  • NLP for phishing email detection
  • Graph neural networks for APT detection
  • Tools: Darktrace, CrowdStrike Falcon AI, Microsoft Sentinel
🔴
AI for Red Teaming
AI-powered offensive security tools that automate reconnaissance and exploitation.
  • AI-generated phishing (spear phishing at scale)
  • Automated vulnerability discovery with ML
  • Pentera / NodeZero — autonomous pentesting
  • AI code assistants writing exploit code
  • LLM-based social engineering simulation
🤖
LLM Security Risks
New attack surfaces created by large language models in applications.
  • Prompt injection attacks on AI systems
  • Data exfiltration via LLM APIs
  • Jailbreaking AI safety guardrails
  • Training data poisoning
  • Indirect prompt injection via documents/emails
📊
AI for SOC Operations
Automating the repetitive work of a Security Operations Center.
  • Alert triage and prioritization
  • Automated incident playbooks
  • Threat intelligence summarization
  • SOAR (Security Orchestration with AI)
  • Natural language querying of SIEM data
🧠
Build AI Security Tools
Practical projects you can build to combine AI with security expertise.
  • AI log analyzer (Python + Claude API)
  • Phishing URL detector with ML
  • CVE summarizer using LLMs
  • Automated pentest report generator
  • Malware classification with deep learning
🎓
Skills to Develop
What you need to learn to be an AI-augmented security professional.
  • Python + scikit-learn / TensorFlow basics
  • Working with APIs (Anthropic, OpenAI, VirusTotal)
  • Prompt engineering for security use cases
  • Understanding ML model limitations (adversarial ML)
  • NIST AI Risk Management Framework

Career Paths

Your options after mastering cybersecurity — both government and private sector in India.

🏛️
Government Jobs (India)
High job security, prestige, and growing cybersecurity roles in Indian government.
  • NIC (National Informatics Centre) — IT & Cyber roles
  • DRDO — Defense cybersecurity research
  • CERT-In — Incident response and threat intel
  • MHA — Cyber crime division (CBI/NIA)
  • NCIIPC — Critical infrastructure protection
  • UIDAI — Aadhaar security
  • RBI IT Department — Banking cyber security
  • State Police Cyber Cells — Cybercrime investigation
  • SSC CGL (Group B) with IT background
  • UPSC — IPS with cyber specialization
💼
Private Sector
High salaries, fast growth, and global opportunities in private cybersecurity.
  • SOC Analyst (Tier 1, 2, 3) — ₹4–20 LPA
  • Penetration Tester — ₹8–30 LPA
  • Threat Intelligence Analyst — ₹10–25 LPA
  • Cloud Security Architect — ₹20–50 LPA
  • CISO (after 10+ years) — ₹50 LPA+
  • Bug Bounty (part time) — Variable
  • Security Consultant — ₹15–40 LPA
  • Malware Analyst — ₹10–25 LPA
🚀
Start Your Own Business
The highest earning potential — build a cybersecurity firm serving SMBs, government, and enterprises.
  • Register OPC / Pvt Ltd company (MCA portal)
  • Get empaneled with CERT-In for audits
  • Offer VAPT services to local businesses
  • Apply for STQC empanelment
  • GEM (Government E-Marketplace) registration
  • Offer security awareness training
  • Managed SOC as a Service (MSSPs)
  • ISO 27001 consulting for SMBs
🎓
Academia & Research
Contribute to the field through research, teaching, and publishing.
  • Pursue M.Tech / M.Sc Cyber Security
  • PhD in cybersecurity-related areas
  • Publish papers at IEEE S&P, USENIX, CCS
  • Teach at engineering colleges
  • Research at IITs, IIITs, CDAC
  • Write cybersecurity books/courses
🌍
Freelancing / Remote
Work from anywhere — Indian talent is in demand globally in cybersecurity.
  • Bug bounty programs (HackerOne, Bugcrowd)
  • Freelance VAPT on Upwork/Toptal
  • Create courses on Udemy, Teachable
  • YouTube cybersecurity channel (Hindi/English)
  • Security blog → consulting leads
  • Write CTF writeups → community reputation
📋
Exam Preparation Tips
How to crack government cybersecurity exams in India.
  • Master networking + OS fundamentals
  • Learn about Indian IT Act 2000 (Amendment 2008)
  • Study CERT-In guidelines and advisories
  • Practice GATE CS exam (for PSU jobs)
  • NIC exam pattern: Technical MCQs + aptitude
  • Keep up with NCSC, CERT-In news
  • Get CompTIA Security+ before applying

function navigateTo(page) {
  document.querySelectorAll('.page').forEach(p => p.classList.remove('active'));
  document.querySelectorAll('.nav-link').forEach(a => a.classList.remove('active'));
  
  const target = document.getElementById('page-' + page);
  if (target) target.classList.add('active');
  
  const link = document.querySelector(`.nav-link[data-page="${page}"]`);
  if (link) link.classList.add('active');
  
  window.scrollTo(0, 0);
  updateAllUI();
}

document.querySelectorAll('.nav-link').forEach(link => {
  link.addEventListener('click', e => {
    e.preventDefault();
    navigateTo(link.dataset.page);
  });
});

function drawMatrix() {
  ctx.fillStyle = 'rgba(5,10,14,0.05)';
  ctx.fillRect(0, 0, canvas.width, canvas.height);
  ctx.fillStyle = '#00ff88';
  ctx.font = '12px Share Tech Mono';

    const char = String.fromCharCode(0x30A0 + Math.random() * 96);
    ctx.fillText(char, i * 16, drops[i] * 16);
    if (drops[i] * 16 > canvas.height && Math.random() > 0.975) drops[i] = 0;
    drops[i]++;
  }
}
